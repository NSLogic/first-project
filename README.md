# TastyTreats

A full-stack menu application built using the MERN stack. The objective is to provide a responsive interface where users can browse categories, search for dishes, and manage a shopping cart with real-time updates.

## Project Features

* **Role-Based Access Control:** Standard users can manage bookmarks and their shopping cart. The owner has a dedicated dashboard to add, update, or delete recipes.

* **Dynamic Interface:** A responsive frontend designed for searching and browsing recipes.

* **State Management:** Real-time cart updates and synced user data.

## Development Notes

I have focused on building a solid foundation for authentication and state management. Managing the communication between the frontend and backend presented several challenges, particularly in keeping the data in sync; however, this process has been instrumental in understanding how to connect the two layers of the application.

The current checkout flow is a simulation. I have opted to hold off on integrating a real payment gateway until the core architecture is stable and performing as expected. I am currently refining the admin dashboard and continuing to optimize how the components interact. This project has served as an effective way to learn how to structure a functional, real-world application.
